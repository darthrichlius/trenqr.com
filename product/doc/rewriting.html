Les règles sur le REWRITING pour le produit TRENqR


Liste des différents formats d'URL pour le produit

TIMELINER

  • /@[_USERNAME] : L'utilisateur veut atteindre la page de l'utilisateur spécifié. Il arrivera sur une page dispatcheuse.
  • /[_USERNAME] : L'utilisateur veut atteindre la page de l'utilisateur spécifié. Il arrivera sur une page dispatcheuse.
  • /[_USERNAME]/[_PAGE_NAME]/w=[_URQID] : L'utilisateur veut atteindre la page de l'utilisateur spécifié selon l'urqid spécifié.
  • /[_USERNAME]/[_PAGE_NAME]/w=[_URQID]&[_UPS1.UPS2.UP3....] : L'utilisateur veut atteindre la page de l'utilisateur spécifié selon l'urqid spécifié + UPS.

AUTRES PAGE

  • /[_ZONE_DU_PRODUIT (PAGE_NAME)]/[_URQID] : L'utilisateur veut atteindre la page de la Tendance spécifiée. Cette page fait aussi office de dispatcheuse. C'est la page génréral des Tendances
  • /[_ZONE_DU_PRODUIT (PAGE_NAME)]/[_URQID]&[_UPS1.UPS2.UP3....] : L'utilisateur veut atteindre la page de la Tendance spécifiée + UPS.


EXEMPLES (Les exemples représentent au mieux la réalité)

TIMELINER

  • /@BartSimpson
  • /BartSimpson
  • /BartSimpson/timeline/w=at-home : L'utilisateur @BartSimpson veut atteindre sa page
  • /BartSimpson/profil/w=visiting : L'utilisateur veut atteindre la page de @BartSimpson.
  • /BartSimpson/profil/w=discovery : L'utilisateur veut atteindre la page de @BartSimpson quand il n'est pas connecté

TREND-PAGE

  • /trend/Hum-donuts.78ae45sq : L'utilisateur veut atteindre la page de la Tendance Hum-donuts. Cette page est une dispatcheuse.
  • /trend/explore/Hum-donuts.78ae45sq : L'utilisateur veut atteindre la page de la Tendance Hum-donuts. Cette page est une dispatcheuse.
  • /trend/manager/Hum-donuts.78ae45sq : L'utilisateur veut atteindre la page de la Tendance Hum-donuts quand l'utilisateur actif est le propriétaire de la Tendance.

AJAX

  • /ajax/profil/w=TMNLR_ARP_ADDRCT&ups=ue-4g369tffvfd : Requete AJAX pour atteindre l'urq spécifié avec comme PAGE : "profil" & UPS : (ueid) "4g369tffvfd".
